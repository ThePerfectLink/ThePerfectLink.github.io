I have since taken over as the lead web dev, and do both frontend and backend work to implement new features on my freetime.

Software

HandTrainer

A Texas Hold 'Em application I made for a friend when he requested from me a way to train his responses when it came to folding or raising given a hand and a seat. Can be customized by altering a file provided so that you can train any given set of acceptable hands with a set of given seats. As of right now though it's limited to 3 types of seats: Early, Middle and Late. soon I will add additional seats and customization for those seats. But that'll have to wait until I have a little more financial security.
